# TaskMap プロジェクト概要

## プロジェクト名
Task Map

## 概要
タスク管理を視覚的に行うためのノードベースのマッピングシステムです。タスクの階層構造を可視化し、依存関係や担当者、期日を管理できます。

## デプロイ先
- 本番環境: https://task.kanazawa-application-support.jp
- デプロイ方法: GitHub Actionsによる自動デプロイ（mainブランチへのプッシュで自動実行）

## 技術スタック
- **React 18** - UIフレームワーク
- **TypeScript** - 型安全性
- **Vite** - ビルドツール
- **ReactFlow** - ノードベースUI
- **Zustand** - 状態管理（永続化対応）
- **Tailwind CSS** - スタイリング
- **Framer Motion** - アニメーション
- **dayjs** - 日付処理

## 主要機能
1. タスクの階層構造の可視化
2. タスク間の依存関係の管理
3. 担当者の割り当て
4. 期日の設定と管理
5. モバイル対応UI（画面幅1024px未満）
6. ドラッグ&ドロップによる直感的な操作
7. Undo/Redo機能
8. タスクのコピー&ペースト
9. 複数選択機能

## 開発環境
- Node.js 20以上
- npm
- 開発サーバー: http://localhost:5173
